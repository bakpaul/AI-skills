The main SKILL.md provides a concise overview. When more detail is needed:

1. **Quick answers** → Use quick-reference.md
2. **Fixing issues** → Use pitfalls.md + debugging.md
3. **Learning concepts** → Use topic-specific files (targets.md, etc.)
4. **Complete examples** → Use examples.md or ../examples/ directory
